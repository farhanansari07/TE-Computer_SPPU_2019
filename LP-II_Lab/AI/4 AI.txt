📘 Theory (Short)

The N-Queens problem is a classic example of a Constraint Satisfaction Problem (CSP) where we place N queens on an N×N chessboard such that no two queens attack each other.

Constraints:

No two queens in the same row
No two queens in the same column
No two queens on the same diagonal

To solve this, we use:

🔹 Backtracking
Place a queen row by row
If a conflict occurs, remove the queen and try another position
🔹 Branch and Bound
Before placing a queen, we check if the position is safe
If unsafe, we skip that branch, reducing unnecessary computation

This makes the solution efficient compared to brute force.

📋 Algorithm (Short)
N-Queens using Backtracking
1.Start from the first row
2.For each column in the current row:
	Check if placing a queen is safe
3.If safe:
	Place the queen
	Move to the next row (recursive call)
4.If all queens are placed:
	Return success
5.If not safe or no solution found:
	Remove the queen (backtrack)
6.Repeat until all possibilities are explored

5. What is the N-Queens problem?

6. What are the constraints in N-Queens?
No same column
No same row
No same diagonal
7. Why don’t we check rows in your code?

Because we place only one queen per row, so row conflict is automatically avoided.

8. How do you represent the board?

Using a 2D matrix:

0 → empty

🔹 Backtracking
9. What is Backtracking?

A recursive technique that:

Tries a solution
If it fails → undo (backtrack)
Try next possibility
10. Why use Backtracking in N-Queens?

Because it efficiently explores all valid configurations without brute-force checking all possibilities.

11. What is the base condition in your program?

When row == n, meaning all queens are successfully placed.

12. What happens during backtracking?

The placed queen is removed and the algorithm tries the next column.

🔹 Branch and Bound
13. What is Branch and Bound?

An optimization technique that eliminates unnecessary branches by checking constraints early.

14. Where is Branch and Bound used in your code?

In the is_safe() function which avoids invalid placements.

15. What is pruning?

Removing invalid possibilities early to reduce computation.

16. What is the approach used?
Recursive Backtracking
Constraint checking using Branch & Bound
