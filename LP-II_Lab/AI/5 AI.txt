📚 Theory (Short)

A chatbot is a program that simulates conversation with users. It is used in customer support, websites, and apps to answer queries automatically.

This practical uses a rule-based chatbot, which works on predefined conditions. It checks user input and gives responses based on keywords like “product”, “order”, or “return”. It does not learn from data but follows fixed rules.

⚙️ Algorithm (Short)
Start the program
Display welcome message
Accept user input
Convert input to lowercase
Check conditions:
If greeting → show greeting message
If product query → show product list
If order → ask order ID and display status
If return → show return policy
If exit → stop program
Else → show default message
Repeat until user exits
Stop
📌 Conclusion (Short)

The elementary chatbot was successfully developed using Python. It can handle basic customer queries like product information, order status, and return policy. This practical demonstrates how rule-based systems work in artificial intelligence for simple interactions.
