Step-by-Step Procedure
========================================================
Step 1: Create Trailhead Playground

Open Salesforce Trailhead website.
Login using Salesforce account.
Go to:
Profile → Hands-On Orgs → Create Playground
Wait for Playground creation.
Click Launch.

========================================================
Step 2: Open Salesforce Setup
Inside Playground click the Gear Icon (⚙).
Select:
Setup

This opens Salesforce Administration Panel.

OBJECT CREATION
========================================================
Step 3: Open Object Manager
In Setup search for:
Object Manager
Open Object Manager.
Click:
Create → Custom Object

========================================================
Step 4: Create Custom Object

Enter the following details carefully:

Field	Value
Label	Book
Plural Label	Books
Object Name	Book
Record Name	Book Name
Data Type	Text

Then click:

Save

This creates the custom object.

CUSTOM FIELDS CREATION
========================================================
Step 5: Create Custom Fields

Inside Book Object:

Fields & Relationships → New

Create the following fields one by one:

Field Name	Data Type
Author Name	Text
Publisher	Text
Price	Currency
Length	Number
Available Copies	Number

For every field:

Select datatype.
Click Next.
Enter field label.
Click Next.
Click Save.

CUSTOM TAB CREATION
========================================================
Step 6: Create Custom Tab

Go to:

Setup → Tabs

Under:

Custom Object Tabs → New

Steps:

Select Object = Book
Choose Tab Style
Click Next
Keep defaults
Click Save

This makes the Books object visible in the app menu.

LIGHTNING APP CREATION
========================================================
Step 7: Create Lightning Application

Go to:

Setup → App Manager

Then:

Click:
New Lightning App
Enter:
Field	Value
App Name	Library Management App
Click Next until Navigation Items page appears.

========================================================
Step 8: Add Navigation Items

From Available Items select:

Books

Move it to Selected Items.

Click:

Next

========================================================
Step 9: Assign User Profiles

Select:

System Administrator

Move it to Selected Profiles.

Click:

Save & Finish

LAUNCH APPLICATION
========================================================
Step 10: Launch App

Open:

App Launcher

Search:

Library Management App

Open the application successfully.

🧠 Basic Questions (Very Important)
❓1. What is Salesforce?

👉 Salesforce is a cloud-based CRM platform used to build applications, manage data, and automate business processes without heavy coding.

❓2. What is CRM?

👉 CRM (Customer Relationship Management) is a system used to manage interactions with customers and store their data efficiently.

❓3. What is cloud computing?

👉 Cloud computing means storing and accessing data over the internet instead of local computers.

❓4. What is Salesforce Lightning?

👉 Salesforce Lightning is a modern UI framework used to build faster and more interactive applications.

🧩 Object & Data Model Questions
❓5. What is an Object in Salesforce?

👉 Object is like a table in database used to store data.

❓6. What is the difference between Standard and Custom Object?
Standard Object	Custom Object
Predefined	User-created
Example: Account	Example: Student
❓7. What is a Field?

👉 Field is like a column that stores specific data (e.g., Age, Email).

❓8. What is a Record?

👉 Record is a row of data in an object.

❓9. What is Object Manager?

👉 It is used to create and manage objects, fields, and relationships.

⚠️ Validation Rules (VERY IMPORTANT)

❓15. What is REGEX?

👉 REGEX is used to match patterns in data (like phone number format).

❓16. Why are validation rules important?

👉 To ensure data accuracy and consistency.

👉 Controls who can see or edit fields.
