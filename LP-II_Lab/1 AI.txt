🔷 THEORY (AI-Oriented)
📌 1. Introduction to Graph Search in AI

In Artificial Intelligence, many problems are solved using state space search.
A problem is represented as:

States (nodes) → possible situations
Edges (links) → transitions between states

The goal is to find a path from:
👉 Initial State → Goal State

📌 2. Breadth First Search (BFS)
🔹 Definition

BFS is an uninformed search algorithm that explores nodes level by level starting from the root node.

🔹 Working Principle
First explores all neighbors of a node
Then moves to next level
Uses Queue (FIFO)
🔹 Properties in AI
Complete: ✅ Yes
Optimal: ✅ Yes (for unweighted graph)
Memory: ❌ High
🔹 Applications
Shortest path problems
Social networks
Robot navigation
📌 3. Depth First Search (DFS)
🔹 Definition

DFS is an uninformed search algorithm that explores nodes deeply first before backtracking.

🔹 Working Principle
Go to deepest node first
Then backtrack
Uses Recursion / Stack
🔹 Properties in AI
Complete: ❌ Not always
Optimal: ❌ No
Memory: ✅ Low
🔹 Applications
Puzzle solving
Game trees
Backtracking problems
📌 4. Key Difference (Short)
Feature	BFS	DFS
Strategy	Level-wise	Depth-wise
Data Structure	Queue	Stack
Optimal	Yes	No
Memory	High	Low
🔷 SMALL ALGORITHMS (EXAM READY)

📌 Algorithm: Breadth First Search (BFS)
1.Start from initial node
2.Mark node as visited
3.Insert node into queue
4.While queue is not empty:
	Remove node from queue
	Visit the node
	Add all unvisited neighbors to queue
5.Repeat until all nodes are visited

📌 Algorithm: Depth First Search (DFS)
1.Start from initial node
2.Mark node as visited
3.Visit the node
4.For each adjacent node:
	If not visited, call DFS recursively
5.Repeat until all nodes are visited

🔷 SHORT CONCLUSION (WRITE IN PRACTICAL)

DFS and BFS are fundamental search algorithms used in Artificial Intelligence for exploring state spaces. BFS guarantees the shortest path but requires more memory, whereas DFS is memory efficient but does not guarantee optimal solutions. Both are widely used in problem solving and graph traversal.

🎯 🔷 BASIC QUESTIONS (Very Important)
1. What is a graph in AI?

A graph is a data structure consisting of nodes (states) and edges (actions/transitions). In AI, it is used to represent state space problems.

2. What is state space search?

State space search is the process of exploring all possible states to find a goal state from an initial state.

3. What is DFS?

DFS (Depth First Search) is an uninformed search algorithm that explores nodes deeply first before backtracking.

4. What is BFS?

BFS (Breadth First Search) is an uninformed search algorithm that explores nodes level by level.

5. Why are DFS and BFS called uninformed search?

Because they do not use any heuristic or prior knowledge about the goal.
