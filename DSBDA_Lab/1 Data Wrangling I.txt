📘 SMALL THEORY (Data Wrangling)

Data wrangling (or data preprocessing) is the process of cleaning, transforming, and organizing raw data into a structured format suitable for analysis and machine learning. Real-world datasets often contain missing values, incorrect entries, and inconsistent formats, which can negatively affect results.

In this practical, the diabetes dataset is used. The dataset is first loaded into a pandas DataFrame. Initial exploration is done using functions like head(), shape, columns, and describe() to understand structure and statistics.

Missing values are identified using isnull(). Since some medical values (like Glucose, BMI) cannot be zero, such invalid values are replaced with NaN. These missing values are then handled using mean imputation.

Further, data normalization is applied using MinMaxScaler to bring all features to a common scale (0 to 1), which improves model performance. Finally, categorical variables like “Outcome” are converted into numeric form for machine learning compatibility.

✅ ADVANTAGES
Improves data quality and accuracy
Handles missing and inconsistent values effectively
Makes data suitable for machine learning models
Normalization ensures fair comparison between features
Reduces errors during analysis
❌ DISADVANTAGES
Time-consuming for large datasets
Mean imputation may reduce data variability
Improper handling can introduce bias
Requires domain knowledge (e.g., knowing 0 is invalid)
Over-processing may remove important information
🎯 CONCLUSION

In this practical, data wrangling techniques were successfully applied to the diabetes dataset. The dataset was cleaned by handling missing and invalid values, and transformed using normalization and type conversion. These steps improved the quality and usability of the data, making it ready for further analysis and machine learning tasks. Proper data preprocessing is essential for obtaining accurate and reliable results.
